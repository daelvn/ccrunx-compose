Usage: ccrunx-compose [-v <version>] [-x] [-h] <command> ...

Merge instances of CCRunX together

Options:
          -v <version>,  Prints the ccrunx-compose version
   --version <version>
   -x, --no-cache        Rebuilds the instance again before running
   -h, --help            Show this help message and exit.

Commands:
   build, b              Build an instance of ccrunx-compose
   up, u                 Run an instance
